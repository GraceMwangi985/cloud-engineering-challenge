# Cloud Engineering Challenge

## 1. What is Cloud Computing?
Cloud computing is the delivery of computing services over the internet.

## 2. What is DevOps?
DevOps is a method that combines software development and IT operations.

## 3. Difference Between Docker and Kubernetes
Docker creates containers while Kubernetes manages containers.

## 4. Why Linux Skills Are Important
Linux is widely used in cloud servers and helps manage systems efficiently.

## 5. Cloud Server Problem

If a cloud server becomes inaccessible, I would check:

1. Internet connection
2. Server status
3. Firewall or security settings
